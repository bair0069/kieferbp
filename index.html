My Development Projects

Suggest-A-Movie PWA

"Suggest-A-Movie" nicknamed "SAM". Is a progressive web application that allows you to search for movies that are like movies you have seen before. The application uses Service Workers, IndexedDB, and the Fetch API to store and retrieve data. As a progressive web application the project is fully responsive and works offline.

Single Page Application

This project is a single page application that is used to find actors and their most popular works.When actors are clicked on, you can see a few of their most popular movies. The project was built mobile-first and is fully responsive.

Media Player

This is a media-player that allows you to play music. It is a fully responsive mobile-first application. All the buttons and controls are functional, and music playback can also be controlled by your keyboard.
